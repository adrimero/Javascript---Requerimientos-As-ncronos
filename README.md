## Objetivo

Implementar funcionalidades interactivas en una aplicación web mediante el uso de eventos JavaScript y solicitudes asíncronas, permitiendo la captura de acciones del usuario, la comunicación con fuentes de datos sin recargar la página y la actualización dinámica de la interfaz.

## Descripción

- Eventos en JavaScript
- Peticiones asíncronas (XMLHttpRequest o Fetch API)
- async / await
- Manipulación del DOM
- Comunicación con APIs o datos externos
